# AI Health Assistant

An AI-powered healthcare assistant developed using Python and Flask.

## Features

* Symptom-based disease prediction
* Health precautions and recommendations
* Diet suggestions
* Voice assistant support
* User-friendly web interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SpeechRecognition
* pyttsx3

## Purpose

This project helps users get basic health guidance based on symptoms and demonstrates the application of AI concepts in healthcare.
